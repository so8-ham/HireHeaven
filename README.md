# Microservice Job Portal

This repository contains **HireHeaven**, a microservice-based job portal.

All documentation, architecture diagrams, and setup instructions are in:

**[job-portal/README.md](./job-portal/README.md)**

Quick start:

```bash
cd job-portal
docker compose up -d
# Configure .env files in services/* — see README
# Start each service + frontend — see README
```
